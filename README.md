We began by conducting exploratory data analysis on the training set, which revealed a significant class imbalance (~84% Adult, ~16% Senior) and several missing feature values. To address this, we engineered clinical risk indicators (such as insulin resistance index HOMA-IR and glucose tolerance ratios) and selected an ensemble of LightGBM, XGBoost, and HistGradientBoosting models. We then ran Optuna to optimize hyperparameters under 5-Fold Stratified Cross-Validation, improving the validation ROC-AUC to a highly competitive 0.7597. Finally, we resolved class imbalance at inference time by calibrating the decision threshold to match the training target prior distribution, generating a highly robust set of test predictions predicting exactly 50 Seniors in our final `submission.csv` file.
